南京工程学院 - 校园网802.1X客户端 - 编译及安装
-------------------------------------------------------------------------------

[从源代码编译]
为方便大家研究讨论，我们已将客户端源代码随编译好的可执行文件一同发布。
进行编译之前需安装的几个开发包如下：
对应Ubuntu/Debian的是：
	sudo apt-get install libpcap-dev libgcrypt-dev
对应Fedora/Redhat的是：
	yum install libpcap-devel libgcrypt-devel

编译客户端的命令为：
	cd src
	autoreconf
	./configure --prefix=$HOME
	make
安装：
	make install
注1：如果没有通过--prefix选项指定安装位置，则默认安装至/usr/local目录
注2：暂时不支持make uninstall

如果您在编译过程中遇到困难，可以在校园网802.1X客户端开发小组讨论板上留言
