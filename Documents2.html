[从源代码编译]
为方便大家研究讨论，我们已将客户端源代码随编译好的可执行文件一同发布。
进行编译之前需安装的几个开发包，
对应Ubuntu/Debian的是：
	sudo apt-get install libpcap-dev libgcrypt-dev
对应Fedora的是：
	yum install libpcap-devel libgcrypt-devel
编译客户端的命令为make

如果您在编译过程中遇到困难，可以在校园网802.1X客户端开发小组讨论板上留言
	http://groups.google.com/group/njit8021xclient?hl=zh-CN
或发送Email至开发小组的公共信箱
	njit8021xclient@googlegroups.com

[源代码维护工作]
项目使用Subversion进行源代码版本管理，如果您熟悉svn可以通过以下命令下载源代码的每日快照
        svn checkout http://gforge.osdn.net.cn/svn/njit8021xclient
如果您没有安装Subversion，也还是可以通过网页浏览器在线查看最新源代码：
http://gforge.osdn.net.cn/svn/njit8021xclient/trunk/
http://gforge.osdn.net.cn/plugins/scmsvn/viewcvs.php/trunk/?root=njit8021xclient
欢迎各路高手指点赐教！
